# enpara-ozet

> [!NOTE]
> AI tarafindan yazilmistir.

Enpara'nin sacma ozet pdf'ini okunur ve anlasilir bir txt'ye cevirir. 

```powershell
git clone https://github.com/caglaryalcin/enpara-ozet.git
cd .\enpara-ozet\
```

enpara-ozet klasorunun path'ine ekstrenin veya extrelerin pdf'lerini kopyalayin. Ardindan, 

```powershell
py.exe .\finance_parser.py
```

Okunakli txt'ler hazir.
